# @ungap/marker

A [marker](https://github.com/whatwg/html/issues/11542) for the Web.

  * `@ungap/marker/pi` for the [ProcessingInstruction](https://developer.mozilla.org/en-US/docs/Web/API/ProcessingInstruction) variant
  * `@ungap/marker/comment` for the new [Node](https://developer.mozilla.org/en-US/docs/Web/API/Node) `MARKER_NODE` (as `13`) variant
